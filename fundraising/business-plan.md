## Product

### OpenClaw Platform (Open Source)
**What it is:** Node.js framework for building personal AI agents
**Key features:**
- Multi-channel messaging (WhatsApp, Telegram, Discord, Signal, etc.)
- Memory system (semantic search, long-term recall)
- Skills marketplace (like WordPress plugins)
- Local LLM support (privacy-first option)
- Cron/automation engine
- Multi-agent orchestration

**Business model:** Free forever (Apache 2.0 license)
**Revenue:** None directly - builds community + brand
**Strategy:** Be the infrastructure everyone builds on (like WordPress, React, Kubernetes)

### AlexBot Consumer Product
**What it is:** Hosted personal AI agent service (OpenClaw-powered)
**Key features:**
- **One-click setup** - no coding, hosted service
- **Pre-built integrations** - Gmail, Calendar, WhatsApp, Telegram, Slack, etc.
- **Skills marketplace** - install capabilities like apps
- **Mobile apps** - iOS/Android for on-the-go
- **Privacy controls** - choose cloud vs local LLM, data retention, etc.
- **Personality customization** - make your agent YOU

**Business model:** Freemium SaaS
- **Free tier:** Basic agent, limited channels, cloud-only
- **Pro ($20/mo):** Unlimited channels, local LLM support, priority support
- **Team ($50/user/mo):** Shared agents, collaboration features, SSO
- **Enterprise (custom):** On-prem deployment, custom integrations, SLA

**Revenue projections:**
- Year 1: 10K users, $1M ARR (avg $8.33/user/mo)
- Year 2: 100K users, $15M ARR (avg $12.50/user/mo)
- Year 3: 500K users, $100M ARR (avg $16.67/user/mo)
- Year 5: 2M users, $500M ARR (avg $20.83/user/mo)
