# Channel Memory

Per-channel context files. Load relevant file when responding to that channel.

## Structure
- `clawders.md` - Clawders WhatsApp group
- `esh-*.md` - esh Group work channels (if needed)
- `dm-{name}.md` - Important individual DM contexts

## When to Update
- New key people identified
- Important ongoing conversations
- Learned behaviors for that channel
- Context that should persist across sessions
